Donald E. Knuth's Algorithm X using the Dancing Links technique back to top
Description

I have implemented this algorithm in both .NET and JavaScript - along with multiple demo applications. The algorithm can be used to solve a class of problems known as "exact cover" - essentially where a fixed number of pieces must all be used once each.

Description

A little sliding puzzle app. Allows a puzzle to be solved manually - or just click the "Solve" button and watch the app solve it for you! The solving algorithm is based on iterative deepening A* (IDA*).
